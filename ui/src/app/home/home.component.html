Open Library

This is the home page for the single page web application of a simple library management system. The menu bar on the left consists of six categories namely; Books, Authors, Publishers, Students, Loans, Admins and provide a direct link for tabular representation of data in each category. The categories are connected to a backend database for efficient data storing and retrieving.
Each category has data manipulation user interfaces provided for insertion, updation, deletion. The project is accomplished using Spring Framework at backend and Angular Framework at frontend development. The idea of this project has been derived and extended from the foundation provided here .
